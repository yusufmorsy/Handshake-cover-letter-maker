Privacy Policy

Last updated: August 2025

1. Data We Collect

  • Job-listing content (title, overview, description) scraped from the active Handshake page, purely to generate your cover letter.
  • User-provided OpenAI API key, stored only in your browser via Chrome’s storage API to authenticate requests.

2. How We Use Your Data

All processing happens locally in your browser. We send only the job content and your API key to OpenAI for cover-letter generation. We do not retain, analyze, or share any of your data on external servers.

3. Data Sharing

We do not sell or transfer any user data to third parties beyond the above approved use cases.

4. Your Choices

You can reset or remove your API key at any time by going to the extension’s Options page. You may also clear all extension data via Chrome’s extension settings.

5. Contact

If you have any questions, please open an issue on GitHub or email support@yourdomain.com.
